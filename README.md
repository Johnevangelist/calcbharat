## ✨ Why CalcBharat?

Most existing financial calculator sites are:
- Paywalled after a few uses
- Loaded with ads
- Force account creation
- Slow and bloated
- Collect and sell your data

**CalcBharat is different:**
- ✅ Completely free, forever
- ✅ No signup, no account needed
- ✅ Zero ads
- ✅ All calculations run in your browser — your data never leaves your device
- ✅ Works offline after first load
- ✅ Mobile friendly
- ✅ India-specific (GST slabs, PPF rules, FY 2024-25 tax slabs)

## 🚀 Getting Started

### Use it online
Visit the live site: **[Johnevangelist.github.io/calcbharat](https://Johnevangelist.github.io/calcbharat)**

### Run locally
```bash
git clone https://github.com/Johnevangelist/calcbharat.git
cd calcbharat
# Just open index.html in your browser — no server needed!
open index.html
```
